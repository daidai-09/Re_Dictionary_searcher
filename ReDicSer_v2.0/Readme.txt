========================================================================
  Re.DicSearcher v2.2
  オリジナル辞書作成・検索システム
========================================================================

この度は「Re.DicSearcher」をご利用いただきありがとうございます。
本ソフトウェアは、自分だけの単語帳を作成・管理・検索できるデスクトップアプリケーションです。
軽量なデータベース(SQLite)を採用し、大量の単語も高速に動作します。

■ 動作環境
Windows 10 / 11 (64bit)

■ インストール・起動方法
本ソフトウェアはインストール不要です。
配布された圧縮ファイル(zip)を解凍し、フォルダ内の「Re_DicSearcher_v2」をダブルクリックして起動してください。

【重要：ファイル構成について】
フォルダ内には以下のファイルが含まれています。これらは必ず「同じフォルダ」に置いてください。

1. Re_DicSearcher_v2.exe	... アプリケーション本体
2. dictionary.db       			... 辞書データファイル（※）
3. Re_DicSearcher_v2.py	... Pythonファイル

※ 「dictionary.db」が見当たらない場合、初回起動時に自動的に作成されます。
※ データをバックアップしたい場合は、「dictionary.db」をコピーして保管してください。

------------------------------------------------------------------------
■ 基本的な使い方
------------------------------------------------------------------------

【1. モードの切り替え】
画面右上の [MODE: READ / WRITE] ボタンで、閲覧モードと編集モードを切り替えます。

  [READ]  閲覧モード：検索と閲覧専用。誤操作を防ぎます。
  [WRITE] 編集モード：単語の登録・編集・削除が行えます。

【2. 単語の検索】
・検索バーにキーワードを入力すると、リアルタイムでリストが絞り込まれます。
・検索バー横のラジオボタンで検索方法を変更できます。
  - In    : 部分一致（〜を含む）
  - Start : 前方一致（〜で始まる）
  - End   : 後方一致（〜で終わる）

【3. 単語の登録・編集】
1. [MODE: WRITE] に切り替えます。
2. 画面右側のフォームに「単語」「発音」「品詞」「意味」「使用例」を入力します。
3. [💾 SAVE] ボタンを押すと保存されます。
   （既存の単語と同じ「単語(Term)」を入力して保存すると、上書き更新されます）

【4. 設定変更 (テーマ・起動画面)】
画面左上の [⚙ CONFIG] ボタンから設定を変更できます。

  - UI Theme            : Dark(ハッカー風) / Light(オフィス風)
  - Startup Splash Screen : Classic(アニメーション) / Modern(静止画ロゴ)

変更後、[SAVE & CLOSE] で保存してください。次回起動時にも反映されます。

------------------------------------------------------------------------
■ その他の機能
------------------------------------------------------------------------

・データのインポート
  左上の [📂 IMPORT] ボタンから、過去のJSON形式の辞書データを取り込むことができます。

・並び替え
  リスト横の [▲] [▼] ボタンで、選択した単語の表示順序を入れ替えることができます。
  （※検索フィルターがかかっていない状態でのみ動作します）

■ トラブルシューティング
------------------------------------------------------------------------
Q. 起動時にセキュリティ警告が出る
A. Windowsの仕様により、未署名のアプリは警告が出ることがあります。「詳細情報」→「実行」を選択してください。

Q. データが保存されない
A. フォルダが「読み取り専用」になっているか、圧縮ファイル(zip)の中から直接起動している可能性があります。必ず「すべて展開」してから実行してください。
