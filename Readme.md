  Re.DicSearcher<br>
  オリジナル辞書作成・検索システム<br>
================================================

この度は「Re.DicSearcher」をご利用いただきありがとうございます。<br>
本ソフトウェアは、自分だけの単語帳を作成・管理できるデスクトップアプリケーションです。<br>

■ 動作環境
Windows 10 / 11 (64bit)

■ ファイル構成
フォルダ内には以下のファイルが含まれています。これらは必ず「同じフォルダ」に置いてください。

1. Re_DicSearcher_v6.exe ... アプリケーション本体
2. dictionary.db         ... 辞書データ（自動生成されます）
3. system.config         ... システム設定ファイル（自動生成されます）
4. icon.ico		...icoファイルです。無くても良いです。
5. Re_DicSearcher_v6.py	...ソースコードです。<br>実行には本体(.exe)があるため通常は不要ですが、中身の確認や改変が可能です。

■ 基本的な使い方
------------------------------------------------------------------------

【1. モードの切り替え】
画面右上の [MODE] ボタン、または [Ctrl+M] でモードを切り替えます。

  [READ]  閲覧モード：検索と閲覧専用。誤操作を防ぎます。
  [WRITE] 編集モード：単語の登録・編集・削除が行えます。

【2. 単語の検索】
・検索バーにキーワードを入力すると、リアルタイムでリストが絞り込まれます。
・[Esc] キーを押すと、検索欄をクリアして全件表示に戻ります。
・検索バー横のラジオボタンで検索方法を変更できます。
  - In    : 部分一致（〜を含む）
  - Start : 前方一致（〜で始まる）
  - End   : 後方一致（〜で終わる）

【3. 単語の登録・編集】
1. [MODE: WRITE] に切り替えます。
2. フォームに情報を入力し、[💾 SAVE] または [Ctrl+S] で保存します。
   ※ 同じ単語でも「品詞」が異なれば、別の単語として登録可能です。
   （例：「Run(動詞)」と「Run(名詞)」を別々に登録できます）

【4. Wiki風リンク機能】
意味や例文の中に、他の単語を [ ] で囲んで入力するとリンクになります。
例： `これは [Python] で作られています。`
→ 閲覧モードでクリックすると、「Python」のページへジャンプします。

■ 設定 (Config)
------------------------------------------------------------------------
画面左上の [⚙ CONFIG] ボタンから設定を変更できます。

・UI Theme
  - Dark  : 黒×緑のハッカー風テーマ（デフォルト）
  - Light : 白×黒のオフィス風テーマ

・Splash Screen
  - Classic : v1.0のレトロなアニメーション起動画面
  - Modern  : v2.0のロゴ中心の静かな起動画面

------------------------------------------------------------------------
■ ショートカットキー一覧
------------------------------------------------------------------------
作業効率を上げるため、以下のショートカットキーに対応しています。

  [Ctrl + F]  検索バーに入力フォーカス（即検索）
  [Ctrl + S]  保存（編集モード時）
  [Ctrl + M]  閲覧/編集モードの切り替え
  [Ctrl + N]  入力フォームのクリア（新規登録準備）
  [Esc]       検索フィルターの解除・クリア

■ トラブルシューティング
------------------------------------------------------------------------
Q. 起動しない / エラーが出る
A. フォルダが「読み取り専用」になっているか、ZIPファイルの中から直接起動している可能性があります。必ず「すべて展開」してから実行してください。

Q. データが消えた？
A. 別の「カセット（DBファイル）」を読み込んでいませんか？ [💾 CARTRIDGE] > [LOAD] から、元の dictionary.db を選択し直してください。
